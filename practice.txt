This file is created for practicing git commands.

Features to practice:
- git add
- git commit
- git log
- git branch
- git checkout
- git merge
- git rebase
- git reset
- git diff

Feel free to modify this file to practice different git workflows!

Version 1.0


Git Practice Lab

1. Initialize repository
	git init

2. Create a file and commit
	echo "Hello Git" > hello.txt
	git add hello.txt
	git commit -m "Add hello.txt"

3. Create a new branch
	git checkout -b feature-branch
	echo "Feature work" >> hello.txt
	git add hello.txt
	git commit -m "Update hello.txt on feature-branch"

4. Merge branch
	git checkout main
	git merge feature-branch

5. View history
	git log --oneline --graph --all

6. Revert example
	git revert <commit-hash>

Notes:
- Replace <commit-hash> with the actual hash from git log.
- Use "main" or "master" depending on your repository default.
